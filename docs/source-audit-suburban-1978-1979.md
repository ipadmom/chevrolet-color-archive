# Chevrolet Suburban 1978–1979 exterior-color source audit

## Decision

The official 1978 and 1979 GM Heritage Vehicle Information Kits each contain a
complete Suburban “Interior and Exterior Color Availability Chart.” The public
solid-color timeline publishes only the boldface primary exterior colors. It
does not infer availability from an adjacent model year.

The 1978 chart is at PDF page 30, printed Suburban Page H, and is dated December
23, 1977. The 1979 chart is at PDF page 32, printed Suburban Page H, and is dated
August 18, 1978. The dot beside Carmine interior trim code XRR1 on the 1979
chart is the item restricted to Silverado. It does not restrict an exterior
color.

## Separate two-tone evidence

The solid-color inventory does not flatten secondary-color rows into standalone
offerings. PDF page 31 of the 1978 kit lists these six primary/secondary pairs:

| Primary code | Secondary code |
|---|---|
| 25 | 20 |
| 43 | 12 |
| 53 | 60 |
| 70 | 12 |
| 71 | 76 |
| 81 | 60 |

PDF pages 33–34 of the 1979 kit separately document these Special Two-Tone ZY3
and Exterior Decor Package ZY5 combinations:

| Primary code and color | Secondary code and color | Hood stripe family |
|---|---|---|
| 18 Charcoal | 17 Mystic Silver | Orange to red |
| 25 Mariner Blue | 26 Deep Blue | Blues |
| 65 Light Camel | 60 Santa Fe Tan | Yellow |
| 71 Dark Carmine Red | 71 Dark Carmine Red | Orange to red |
| 81 Cordova Brown | 60 Santa Fe Tan | Yellows |
| 86 Midnight Black | 17 Mystic Silver | Orange to red |

These rows remain a separate paint-scheme evidence class so that primary and
secondary placement is not confused with standalone color availability.
