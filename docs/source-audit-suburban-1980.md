# Chevrolet Suburban 1980 exterior-color source audit

## Decision

The official 1980 GM Heritage Vehicle Information Kit contains a complete
Suburban “Interior and Exterior Color Availability Chart” at PDF page 28,
printed Suburban Page H. The chart is dated August 3, 1979. The public
solid-color timeline publishes the 15 boldface primary exterior colors and does
not convert any lightface secondary row into a standalone offering.

The dot-marked Silverado-only restriction appears on interior trim code LRR1.
It does not restrict an exterior color. The separately sourced 1980 Woodland
Green SEO 9V5 record remains a restricted specialty-paint subset and is not
used to expand this general chart.

## Separate two-tone evidence

PDF page 30, printed Suburban Page J, documents six optional Special Two-Tone
ZY3 and Exterior Decor Package ZY5 combinations. Primary color covers the roof
and the body above and below the moldings; secondary color fills the area
between the moldings.

| Primary code and color | Secondary code and color | Hood stripe family |
|---|---|---|
| 12 Frost White | 71 Dark Carmine Red | Yellow to orange |
| 18 Charcoal (Metallic) | 17 Mystic Silver (Metallic) | Orange to red |
| 30 Nordic Blue (Metallic) | 12 Frost White | Blues |
| 62 Dark Camel (Metallic) | 60 Santa Fe Tan | Brown to gold |
| 71 Dark Carmine Red | 12 Frost White | Orange to red |
| 73 Cardinal Red | 71 Dark Carmine Red | Orange to red |

These rows remain a separate paint-scheme evidence class so that component
placement is preserved and no component is inferred to be a standalone color.
