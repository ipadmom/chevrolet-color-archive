# Camaro 1976–1981 photograph audit

This review adds four rights-checked Wikimedia Commons photographs as
interpretive aids. Two are reviewed illustrations and two remain clearly
marked candidates. None is evidence that a factory color was available, that
the car retains its original paint, or that a photographed hue matches a paint
code.

The crawler and manual review checked exact model year, visible body color,
source-page rights metadata, source-file digest, and downloaded bytes.
Published copies were re-encoded through the archive's pinned
metadata-stripping sanitizer. The complete creator, license, source digest,
published digest, dimensions, caveat, and transform record is preserved in
[`public/vehicle-photos/attribution.json`](../public/vehicle-photos/attribution.json).

## Unresolved 1978 Yellow, Orange search

The official 1978 chart restricts code 34 `Yellow, Orange` to Z28. Forty
Wikimedia Commons candidates were reviewed, but none could be assigned to code
34 without guessing. Exact-year yellow Z28 photographs do not visually
distinguish code 34 from code 51. An exact-title Flickr lead was rejected
because it was all-rights-reserved. The archive therefore publishes no code 34
photograph yet.
