# Chevrolet Suburban 1984–1985 exterior-color source audit

## Decision

The official 1984 and 1985 GM Heritage Vehicle Information Kits each contain a
complete C/K Suburban “Interior and Exterior Color Availability Chart” at PDF
page 31, printed C/K Suburban Page H. The 1984 chart is dated February 1984; the
1985 chart is dated August 1984. Each chart publishes the same ten primary
colors and the same 23 ZY5 combinations.

The 1985 PDF also contains a later chart at PDF page 67 dated September 1985.
That is a different model-year revision and is not used as 1985 evidence.
