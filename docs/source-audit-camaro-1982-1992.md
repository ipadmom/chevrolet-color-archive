# Camaro 1982–1992 official color audit

This audit covers every exterior-color chart and model or package application
grid in the official GM Heritage Camaro kits for 1982 through 1992. It
publishes 108 year-specific chart listings, normalized into 32 exact-name
timeline rows. A row is `listed` when the complete controlling grid supports
the color across the audited model set. A row is `restricted` when the source
limits it to a model or package, excludes it from a model or package, or when
two dated official sources conflict.

The audit unions mutually exclusive model grids so that a color offered on any
Camaro model remains visible, while preserving the application restriction.
It does not infer a paint from brochures, surviving cars, photographs, dealer
advertising, or a repeated code in another year.

## Source reconciliation

The 1985 factory exterior-color specification lists code 60 Light Brown
Metallic, but the January 28, 1985 Dealer Order Guide omits it from every model
and package grid. The archive keeps that official specification listing and
marks the conflict.

The April 1985 publication for the 1986 model year lists codes 60 Light Brown
Metallic and 66 Copper Metallic. The revised December 2, 1985 Dealer Order
Guide omits both. The archive keeps both initial-chart listings and marks each
as absent from the later guide.

The 1987 Color & Trim Selections chart confirms the complete ten-color set for
Sport Coupe and LT. The separate California RS chart says that its displayed
combinations are the only ones available and lists only Black, Bright Red, and
White. The archive therefore marks the other seven colors as unavailable on
California RS. The kit does not provide a complete Z28 or IROC-Z application
grid, so those package applications remain unresolved.

## Exclusions

The audit excludes stripe and decal colors, lower-body accents, wheel colors,
convertible-top colors, Heritage Appearance Package paint layouts, and
multi-color scheme diagrams. Those records can qualify a solid-color
application, but they do not create another solid exterior-color listing.

## Normalization cautions

Canonical timeline rows preserve exact source labels inside each year cell.
The interface normalizes punctuation and word order only for row grouping.
Important changes remain visible:

- Code 98 is `Blue, Bright (Met)` in 1989–1990 and `Blue, Ultra (Met)` in 1991.
- Dark Red Metallic changes from code 74 in 1986–1989 to code 75 in 1990–1992.
- `White`, code 40, in 1986–1990 is not merged with `White, Arctic`, code 10,
  in 1991–1992.
- `Green Dk, Gray (Met)` is preserved exactly for 1992 code 18.
- An identical code never proves identical paint across model years.
