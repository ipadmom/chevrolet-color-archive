# 1976–1981 Camaro exterior-color source audit

This audit reviews the complete Camaro exterior-color tables in six dedicated
GM Heritage kits. The public matrix contains 82 year-specific chart listings:
81 listed colors and one trim-restricted color. The restricted record is 1978
`Yellow, Orange`, code 34, which appears as an exterior paint only in the Z28
table.

The public timeline expands the chart abbreviation `(Met)` to `Metallic` and
regularizes inverted adjective order only in canonical row names. Year cells
retain the exact chart labels below. A shared paint code does not merge
different names or formulations.

## Promotion rule

A color enters the public matrix only when it appears in the complete cited
Camaro table or, for 1978 code 34, in the complete Z28 exterior-paint column.
Rally Sport pairs, Z85 applications, stripe systems, and Z28 scheme labels stay
in source notes. They are not promoted as separate solid-color records.
