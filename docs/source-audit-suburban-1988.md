# Chevrolet Suburban 1988 exterior-color source audit

## Decision

The official 1988 GM Heritage Vehicle Information Kit contains two governing
R/V Suburban color pages. PDF page 29, printed Page H and dated September 1987,
contains the same-code primary entries and the shared ZY1/ZY3 two-tone chart.
PDF page 30, printed Page I and dated February 1988, contains the ZY5 Exterior
Decor Package chart. The public solid-color timeline publishes only the ten
same-code primary entries from page 29.

## Verification

- The retained PDF has 31 pages, is 1,073,435 bytes, and has SHA-256
  `8dd6db35aa1e4901941ac0ef4cdd5bb509bbf30be651aafeb23307c66e5b0e7e`.
- The artifact ledger records source ID
  `gm-heritage-1988-chevrolet-suburban`, HTTP 200, and `application/pdf`.
- PDF pages 29 and 30 were rendered at 400 DPI and checked row by row.
- All ten same-code primary entries, 49 shared ZY1/ZY3 pairs, and 63 printed
  ZY5 rows were visually reconciled.
