# C1 Corvette color-table source audit

This audit covers the dedicated 1953 through 1962 Corvette PDFs in the
official GM Heritage Vehicle Information Kit directory. It records only what
the cited documents support.

## 1953

The 25-page dedicated kit was reviewed page by page. It contains an Automobile
Manufacturers Association specification questionnaire dated November 15, 1953,
but no exterior-color table. The archive therefore leaves 1953 unverified.
This does not mean that no exterior color was available.

## 1954

The `1954 COLORS` table appears at PDF page 4, printed page 23.

| Exterior | Code | Estimated quantity |
|---|---|---:|
| Polo White | Not stated | 3,230 |
| Pennant Blue | Not stated | 300 |
| Sportsman Red | Not stated | 100 |
| Black | Not stated | 4 |

The table expressly describes those quantities as estimates rather than
Chevrolet records. It also says that additional exterior colors may have been
used and names Metallic Green and Metallic Bronze as possibilities that were
not confirmed by the records discussed there. The archive publishes the four
table rows as restricted evidence and does not promote the two possible colors.

## 1955

The `1955 COLORS` table appears at PDF page 3, printed page 25.

| Code | Exterior | Estimated quantity |
|---:|---|---:|
| 567 | Polo White | 325 |
| 570 | Pennant Blue | 45 |
| 573 | Corvette Copper | 15 |
| 596 | Gypsy Red | 180 |
| 632 | Harvest Gold | 120 |

The same page says the quantities are estimates, that surviving Chevrolet
records do not document color usage, and that the exterior-color history is
partly uncertain. It reports that Pennant Blue was discontinued in April 1955
and characterizes Gypsy Red and Corvette Copper as colors thought to have been
offered afterward. All five rows are therefore published as restricted
evidence, with the page-specific qualification retained.

## 1956

The `1956 COLORS` table appears at PDF page 4, printed page 27.

| Exterior | Code | Quantity |
|---|---|---:|
| Onyx Black | Not stated | 810 |
| Aztec Copper | Not stated | 402 |
| Cascade Green | Not stated | 290 |
| Arctic Blue | Not stated | 390 |
| Venetian Red | Not stated | 1,043 |
| Polo White | Not stated | 532 |

Unlike the two preceding tables, the source says the exterior quantities come
from Chevrolet records, total exactly 3,467 cars, and make it unlikely that
other exterior colors were possible in 1956. These six rows are therefore
published as listed rather than restricted. The table does not map individual
paint codes to the color names, so the archive records each code as not stated.

## 1958

The dated Chevrolet color-combination chart appears at PDF page 12, printed
`CORVETTE SUPPLEMENT - P49`, dated November 29, 1957. A supplemental `1958
COLORS` production table appears at PDF page 6, printed page 31.

| Exterior | Code | Quantity |
|---|---|---:|
| Charcoal | Not stated | 1,631 |
| Snowcrest White | Not stated | 2,477 |
| Silver Blue | Not stated | 2,006 |
| Regal Turquoise | Not stated | 510 |
| Panama Yellow | Not stated | 455 |
| Signet Red | Not stated | 1,399 |
| Black | Not stated | 493 |
| Silver | Not stated | 193 |

The listed quantities total 9,164. The production table accounts for the four
remaining cars as non-standard color, combination, or primer, reconciling to
9,168 production. It also says Charcoal is believed to have been replaced by
Black during the model year. The direct Chevrolet chart independently supports
the eight standard colors, so their availability is published as listed.

The Chevrolet `EXTERIOR - INTERIOR COLOR COMBINATIONS` chart appears at PDF

1960. A supplemental `1960 COLORS` production table appears at PDF page 28,
printed page 35.

| Exterior | Code | Quantity |
|---|---|---:|
| Tuxedo Black | Not stated | 1,268 |
| Tasco Turquoise | Not stated | 635 |
| Horizon Blue | Not stated | 766 |
| Honduras Maroon | Not stated | 1,202 |
| Roman Red | Not stated | 1,529 |
| Ermine White | Not stated | 3,717 |
| Sateen Silver | Not stated | 989 |
| Cascade Green | Not stated | 140 |

The quantities total 10,246. The source accounts for 15 additional cars as a
non-standard color, combination, or primer, reconciling to 10,261 production.
The dated Chevrolet chart supports the eight standard colors, so they are
published as listed. The source describes 1960 Cascade Green as a different
metallic paint from the 1956–1957 color of the same name, so the timeline keeps
them as separate records.

## 1962

The Chevrolet `EXTERIOR - INTERIOR COLOR COMBINATIONS` chart appears at PDF
page 14, printed `CORVETTE-5`, dated October 1961. A supplemental `1962 COLORS`
table appears at PDF page 6, printed page 39.

| Exterior | Code | Quantity |
|---|---|---:|
| Tuxedo Black | Not stated | Not stated |
| Fawn Beige | Not stated | 1,851 |
| Roman Red | Not stated | Not stated |
| Ermine White | Not stated | Not stated |
| Almond Beige | Not stated | 820 |
| Sateen Silver | Not stated | Not stated |
| Honduras Maroon | Not stated | Not stated |

The source says individual cars were not coded by exterior color and gives
quantities only for Fawn Beige and Almond Beige. The dated Chevrolet chart
nevertheless identifies the seven standard body colors, so they are published
as listed. The supplemental prose says other exterior colors were built and
gives Cadillac Royal Heather Amethyst as a special-use example. That prose-only
non-standard color remains an unverified candidate rather than a matrix row.

## Promotion rule

A color enters the public matrix only when it appears in a completely reviewed
table. Possible colors discussed only in prose remain unverified candidates.
An interpretive screen swatch is never documentary evidence.
