# Chevrolet Suburban exterior-color audit, 1995-1999

## Audit scope and transcription rules

This is a visual transcription of the retained official Chevrolet Suburban order-guide scans. Every cited chart page was rendered at 300 dpi and inspected visually. The PDFs have no useful text layer. OCR was used only as a reading aid and never as the controlling transcription.

Spacing and capitalization are made legible, but the printed color-name word order, abbreviations, punctuation, `U` and `L` code suffixes, scheme codes, restriction markers, duplicate rows, stripe variants, and wheel-flare labels are preserved. A slash separates the two printed color fields. `Interior` records every printed `X`, in the chart's column order: Neutral or Beige, Blue, Gray, Red. A blank interior cell in the source remains absent here.

The governing distinction is:

- `ZY1`: one-color or solid application. These rows define the standalone model-year palette.
- `ZY2`: conventional two-tone. Color Code 1 is the upper/body role and uses `U`; Color Code 2 is the lower role and uses `L`.
- `ZY4`: deluxe two-tone, with a separate printed stripe-color field. It is present in the 1995-1997 Suburban charts and absent from the 1998-1999 Suburban option lists and charts.

A code printed only in a `ZY2` or `ZY4` Color Code 2 field is scheme evidence, not a standalone-color row. The same numeric code with `U` and `L` suffixes represents different ordering roles and is not flattened. Duplicate code pairs with different stripe or interior variants remain separate rows.

## Tahoe corroboration, never inference

The already audited GMT400-era Tahoe charts corroborate the Suburban transcription but do not supply any Suburban row:

- 1995: the Suburban 10-row ZY1 palette matches Tahoe's 10 standalone codes. The 22 Suburban ZY2 rows match Tahoe's 2-door conventional set; Tahoe's separate 4-door Gray Gunmetal rows are not imported into Suburban. Suburban ZY4 remains Suburban-specific evidence.
- 1996: the Suburban 10 ZY1 rows and 24 ZY2 rows match Tahoe's independently printed 10 and 24. Suburban ZY4 remains separately controlled by the Suburban page.
- 1997: the Suburban 9 ZY1 rows and 27 conventional ZY2 rows match Tahoe's conventional set. Tahoe's separate BYP Sport rows are not imported. Suburban ZY4 remains separately controlled by the Suburban page.
- 1998: the 9 Suburban ZY1 rows match Tahoe's 9 standalone codes. The Suburban chart has only 21 ZY2 rows, while Tahoe has a larger body-style and BYP corpus. No Tahoe-only scheme is imported. The restriction language also differs, confirming that model-specific charts must control.
- 1999: the 10 Suburban ZY1 rows and 34 Suburban ZY2 rows match Tahoe's conventional code-pair set. Tahoe BYP rows are not imported. The Suburban `(a)` restriction is `N/A B71, L65 Eng or 4-Wheel Drive`, which is not replaced with Tahoe's body-style-specific restriction language.

## Printed-name and role anomalies that normalization must preserve

- `55U` Autumnwood is a standalone ZY1 color in 1995 and 1996. In 1997 and 1998, only `55L` appears in scheme rows; there is no 1997 or 1998 `55U` standalone row.
- 1998 prints standalone `39U` as `INDIGO, BLUE (Met)` but its ZY2 rows as `BLUE, INDIGO (Met)`. The code identity can normalize, but the literal labels should remain available.
- 1998 prints `GREEN, EMERALD (Met) / Beige Med Mystique (Met)` for `43U/65L`, reversing the usual Mystique word order. Preserve the literal scheme label.
- 1998 prints `BLACK, ONYX / Red, Victory (Met)` in ZY2 even though controlling ZY1 prints `RED, VICTORY` without `(Met)`. Standalone finish follows ZY1.
- 1999 prints standalone `RED, CARMINE DK, (Met)` but one scheme secondary as `Red, Dk Carmine (Met)`. Preserve both literal forms under the same numeric identity.
- 1999 prints `RED, VICTORY` without `(Met)` in ZY1 but `Red, Victory (Met)` in the restricted `41U/74L` scheme. Standalone finish follows ZY1.
- 1995-1997 duplicate ZY4 pairs are real stripe/interior variants and must not be deduplicated solely on model, year, scheme, Code 1, and Code 2.

## Handwritten and scan annotations

- The governing color charts are printed order-guide tables. No handwritten notation was used to create or modify any color row.
- The 1999 order-guide option pages immediately before the charts contain handwritten dealer/user notes near tire, trailering, seat, and skid-plate options. Several are partly illegible. They are unrelated to the printed color charts and are retained only as this annotation.
- Stray pen marks, punched-binder shadows, scan abrasions, and black edge bars appear throughout the source. They are not interpreted as data.
- The 1996 ZY4 wheel-flare cell for `96U/41L` visibly reads `Silve`; the terminal `r` is not visible in the scan. This audit preserves that literal scan condition and does not use it to create a different color identity.
- The 1997 footer says `^ Indicates Change`. Carets are recorded as change annotations, not availability restrictions.
- The 1998 footer says `Shading Indicates Change`. No chart row was promoted, removed, or restricted based solely on scan shading.

## Integration rule

For each model year, publish only the ZY1 table as the primary standalone color list. Store ZY2 and ZY4 in normalized paint-scheme tables with ordered components, interior compatibility, stripe, wheel-flare value, restriction marker/text, source ID, PDF page, printed order-guide page, and revision date. Never infer a Suburban row from Tahoe, never promote a lower `L` code without a same-year Suburban ZY1 row, and never flatten duplicate ZY4 stripe variants.
