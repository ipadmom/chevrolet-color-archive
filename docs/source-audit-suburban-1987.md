# Chevrolet Suburban 1987 exterior-color source audit

## Decision

The official 1987 GM Heritage Vehicle Information Kit contains the complete
R/V Suburban “Interior and Exterior Color Availability Chart” at PDF page 39,
printed R/V Suburban Page H. The chart is dated February 1987. The public
solid-color timeline publishes the ten same-code primary entries. The 33
shared ZY1/ZY3 pairs and 23 ZY5 combinations remain separate paint-scheme
evidence and do not create standalone availability rows.
