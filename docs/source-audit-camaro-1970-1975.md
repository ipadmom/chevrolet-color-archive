# 1970–1975 Camaro exterior-color source audit

This audit reviews the complete Camaro exterior-color charts in six dedicated
GM Heritage kits. The public matrix contains 93 year-specific chart listings.
All 93 are listed. No chart in this block imposes a color-level special-order
restriction.

The public timeline merges only identical marketing names within this audited
block. A repeated numeric code or a similar color family is not enough to merge
two records.

## Chart annotations

The 1973–1975 charts append `C/O` to carryover colors. `C/O` is a status note,
not part of the marketing name, so it is omitted from the canonical row name
but retained in the year-specific chart label. Those charts also permit
nonrecommended solid exterior and interior pairings through the ZP2 override
while retaining vinyl-roof limits. Neither annotation changes a color row from
listed to restricted.

The source literals remain controlling. Canonical rows preserve 1975 `White`,
`Cream-Beige`, and `Persimmon Metallic` rather than substituting aliases from
broader passenger-car material. Their year cells retain the printed `C/O` and
`Met.` forms.

## Promotion rule

A color enters the public matrix only when it appears in the complete cited
Camaro chart. The review found no prose-only or special-order colors for
promotion in this block. Similarity to a prior year, a passenger-car paint
list, or an interpretive swatch is not enough.
