# Chevrolet Suburban 1990-1994 official color-table audit

This is a read-only visual transcription of the locally retained official GM Heritage PDFs. It is a tracked source audit. All page numbers below distinguish PDF page from the page number printed in the source.

Conventions:

- `I=` records the interior-color columns marked by a dot or `x` on that exact row.
- In 1990-1992, a dot is a recommended combination and a blank is available unless the page states otherwise. It is not an exclusivity marker.
- In 1993-1994, `x` marks the combinations the chart says are the only available combinations.
- `restricted` records a source asterisk or boldface restriction. It does not remove the row globally.
- A multi-color scheme component is not a standalone-color finding.
- Names and code anomalies are preserved as printed.

## 1992 C/K Suburban

### Evidence classification and locators

- A governing same-code primary chart is present: ZY1 Solid Color, PDF p20, printed p19. It contains 22 exact same-code rows because optional D85 stripe and interior recommendations create repeated primary colors. There are ten distinct primary exterior colors.
- ZY2 Conventional Two-Tone: PDF p21, printed p20.
- ZY3 Special Two-Tone: PDF p22, printed p21.
- ZY4 Deluxe Two-Tone: PDF p23, printed p22.

- Bold rows are printed `N/A K1500 w/B71 or K2500 models`.
- The kit prints no revision date on these pages, and the PDF metadata has no creation or modification date.
