# Current-model vehicle profile artwork

Updated: 2026-07-22

## Purpose

The model index uses original Chevrolet side-profile SVG artwork. The current-model profiles depict an explicit 2026 U.S. representative configuration instead of inferring a generic shape from a broad class such as “SUV,” “pickup,” or “van.”

The vectors preserve the archive’s selected exterior paint while keeping glass, trim, lamps, tires, and wheels independent. Each profile has its own outer silhouette, greenhouse, wheelbase, overhangs, fascia, lamp placement, door cuts, and distinguishing body details.

## Import Archive comparison

[Import Archive’s Toyota model index](https://importarchive.com/toyota) is the visual-density benchmark. Its model-list artwork is a set of dedicated 140 by 64 pixel JPEG thumbnails, with separate larger raster artwork on generation pages. This project does not copy, trace, recolor, download, or redistribute that artwork. Chevrolet profiles are original vectors based on factual vehicle proportions and Chevrolet’s own current-model references.

Additional benchmark pages inspected:

- [2025 Toyota Camry](https://importarchive.com/toyota/camry/2025-)

Owner manuals remain useful for exact loading, trailering, tire, and configuration definitions, but they are not treated as a substitute for exterior-color evidence. Fleet guides, brochures, order guides, and the current Chevrolet trim/360° pages control the visible vehicle configuration. The SVGs use official dimensions and multiple references while remaining original drawings rather than copies of Chevrolet or Import Archive artwork.

The official [GM Manuals and Guides portal](https://experience.gm.com/support/vehicle/manuals-guides)
returns exactly the same 18 current 2026 model entries used by the archive. The
machine-readable
[`chevrolet-current-manual-references.json`](../data/catalog/chevrolet-current-manual-references.json)
map records each portal model ID, direct owner-manual PDF, live content type and
byte length, plus the Vehicle Reference Information document where Chevrolet
publishes one. Sixteen unique owner manuals cover all 18 models because
BrightDrop 400/600 and Tahoe/Suburban each share an official manual. The Low
Cab Forward profile is matched specifically to the 6500XD/7500XD 6.7L manual,
not a different LCF engine family.

For current paint, the [GM Electronic Order Guide](https://www.gmfleetorderguide.com/)
is stronger than the owner manuals because it supplies RPOs, WA touch-up
identities, trim compatibility, SEO rows, order restrictions, and official
swatch filenames. Its 26 active 2026 Chevrolet entries reduce to 17 guide IDs
for the 18 archive profiles because BrightDrop 400 and 600 share one guide.
Exact IDs, endpoints, omissions, and source-literal conflicts are preserved in
the
[`current-model-order-guide-reconciliation.json`](../data/audits/current-model-order-guide-reconciliation.json)
audit.

## Rendering requirements

Every approved current-model profile must:

1. Resolve only for its documented latest-generation year band.
2. Expose the exact profile ID, representative year, and representative variant in SVG data attributes.
3. Use a unique outer silhouette. Related vehicles such as Tahoe and Suburban, BrightDrop 400 and 600, and Silverado 1500, HD, and EV cannot share paths.
4. Keep paint dynamic through separate base, highlight, and shadow layers.
5. Remain recognizable at the model index’s thumbnail size.
6. Fall back to the archive’s year-aware historic renderer outside the latest-generation band.

The automated coverage gate verifies all 18 current nameplates, latest catalog
years, unique geometry, related-model separation, official Chevrolet reference
links, and detailed-renderer metadata. For every profile whose Fleet Guide page
omits wheelbase, it also recomputes the source wheelbase-to-length ratio and
compares that value to the SVG calibration target.
